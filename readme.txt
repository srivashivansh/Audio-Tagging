	    INFERENCE
====================================

TASK-1: Audio event detection
-----------------------------

The submitted csv file should be in the following format with header:
<filename-without-extension>,<event-music/speech>,<onset>,<offset>

E.g.:
filename,event,onset,offset
test_sample-2,Music,0.85,3.65
test_sample-2,Speech,5.95,9.11
test_sample-3,Music,7.33,9.42
test_sample-5,Speech,7.49,9.64
test_sample-6,Speech,3.18,6.16



TASK-2: Audio Tagging
---------------------

The submitted csv file should be in the following format with header:
<filename-without-extension>,<music-0/1>,<speech-0/1>

E.g.:
filename,Music,Speech
test00,0,1
test01,1,1
test02,0,0
